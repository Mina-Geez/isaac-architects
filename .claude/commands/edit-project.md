Edit a project in the content engine.

1. Open `content/projects/<slug>/index.md` (ask which project if not given).
2. Apply the requested changes — copy (lead/body), metadata (location, type, area, status, category, order, featured), or gallery (reorder `[[gallery]]` entries, fix `alt`, add/remove images in the folder).
3. Run `python build.py` and report what changed.
4. Tell me to run `/publish` when ready.
